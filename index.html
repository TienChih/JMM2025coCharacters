Background

Definition: A lattice is a partially ordered set closed under joins (\(\vee\)) and meets (\(\wedge\)).


Definition: Given a finite lattice \((P, \leq)\), a transfer system is an order relation \(R\) which refines \( \leq \), includes all identities, and is closed under pullback.

\[x\leq z, y~R~z \Rightarrow x\wedge y~R~y\]

Definition: Given a category \(C\), an weak factorization system on \(C\) is a pair of collections of morphisms \((L, R)\), such that:

  • For each morphism \(f\) of \(C\), we have that \(f=pi\) for some \(i\in L\), \(p\in R\).
  • \(L= ^\bot R\) and \(R=L^\bot\).

When \(\lambda\) is unique, we say \((L,R)\) is an orthogonal factorization system.

Theorem (FOOQW, 2022): Let \(P\) be a finite lattice, and let \(R\) be a transfer system on \(P\). Then there is a unique weak (orthogonal) factorization system \((L, R)\) on \(P\).

Theorem (FOOQW, 2022): Let \(P\) be a finite lattice, and let \(R\) be a transfer system on \(P\). Then there is a unique weak (orthogonal) factorization system \((L, R)\) on \(P\).


Theorem (FOOQW, 2022): Let \(P\) be a finite lattice. Then \[R\longleftrightarrow (L, R)\] gives an isomorphism between the poset of transfer systems on \(P\) and the poset of weak factorization systems on \(P\).

Definition: Given a finite lattice \(P\), we define the characteristic function \(\chi\) to be a map \[ \begin{aligned} \chi\colon \mathrm{Tr}\ P&\longrightarrow \mathrm{End}\ P\\ R&\longmapsto \chi^R. \end{aligned} \] where \(\chi^R\) is the endomorphism which maps \(\chi^R\colon x\mapsto \min\{y\mid y~R~x\}\).

Definition: A transfer system \(R\) on a lattice \(P\) is called saturated when \(x ~R~ y\leq z\) and \(x ~R~ z\) implies \(y ~R~ z\).

Theorem (BHKOPST): Let \(P\) be a finite lattice, then \(\lambda:\mathrm{sat}(P)\to \mathrm{End}^o(P)\) is bijective.


Theorem (BHKOPST): The fibers of \(\chi:\mathrm{Tr}(P)\to \mathrm{End}^o(P)\) have a maximal element, which are exactly the saturated transfer systems of \(P\).

Definition: Given a finite lattice \(P\), we define the cocharacteristic function \(\lambda\) to be a map \[ \begin{aligned} \chi\colon \mathrm{Tr}\ P&\longrightarrow \mathrm{End}\ P\\ R&\longmapsto \lambda^R. \end{aligned} \] where \(\lambda^R\) is the endomorphism which maps \(\lambda^R\colon x\mapsto \min\{y\mid x\le y~R~1\}\).

Definition: A transfer system \(R\) on a lattice \(P\) is called cosaturated if \(R\) is generated by the relations to \(1\).

Definition: For a category \(\mathbf{C}\) with terminal object 1, we say an orthogonal factorization system \((L, R)\) is a reflective factorization system if \(R/1\) is a reflective subcategory of \(\mathbf{C}\), that is the inclusion function \(\Phi:R/1\to \mathbf{C}\) has a left adjoint \(\Psi\).


Theorem (coCharacters): Let \(P\) be a finite lattice and let \(R\) be a transfer system of \(P\). Then the factorization system \((L, R)\) is reflective if and only if \(R\) is cosaturated.

Definition: For a category \(\mathbf{C}\) with terminal object 1, we say an orthogonal factorization system \((L, R)\) is a reflective factorization system if \(R/1\) is a reflective subcategory of \(\mathbf{C}\), that is the inclusion function \(\Phi:R/1\to \mathbf{C}\) has a left adjoint \(\Psi\).
